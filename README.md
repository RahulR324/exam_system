# 📘 Online Examination System

A full-stack **Online Examination System** developed using **React.js**, **HTML**, **CSS**, **JavaScript**, **CodeIgniter 4**, and **MySQL**.

This platform allows students to securely attend online exams through a responsive mobile-friendly interface while enabling administrators to manage exams, students, questions, and results efficiently.

---

# 🚀 Features

## 👨‍🎓 Student Module

* Secure Student Login
* Mobile Responsive Exam Interface
* Real-Time Exam Timer
* Auto Submit on Time Expiry
* Previous / Next Navigation
* Live Progress Tracking
* Result Generation
* JWT Authentication

---

## 👨‍💼 Admin Module

* Admin Authentication
* Create & Manage Exams
* Add Questions
* Upload Student Data
* View Student Results
* Edit/Delete Exams
* Dashboard Management
